#  前言

大一下开学了一个月，现在我在西安北站等待前往重庆的高铁。也算是在平淡本科生活的一次越狱。之前也考虑写过一点点周记，不过因为事情太多很难每周有足够的时间来留给写周记，因此把周记的形式改为月记。一方面也是因为笔者现在的专业，确实没有什么可以输出的专业课，一些无聊的水课的文稿也大多是 LLM 生成的产物。为了不被 AI 彻底吞噬我的写作表达能力，因此开一个月记的专栏来记录这一个月发生的事情。

# 学校

关于学校这部分，的确是有些平淡。大概写一点流水账的记录吧。二月末很早的回到学校，一方面是为了早点回学校熟悉环境，另一方面也是提前回来看看出租屋的事情。不过最后因为其他的因素也就暂时搁置了。后来就是每周无聊的上课，这学期跨选了新思想，老师每日水课已成为常态。不过这样也让我被迫起早八做一些感兴趣的事情避免了到宿舍就睡觉的脑雾状态。幸运的是，这学期的高数课相较于上学期有意思了许多，常争鸣老师的纯板书形式的授课反而更能让我保持注意力，莫名有一种回到预科二王立周老师数学课的恍惚感（？）。运动会这方面就是和去年一样，仍然是八字大绳和集体大绳。和去年的成绩大概率也差不多。不过很惊喜的是，这个刚刚组队不到两周的仲英草台班子队伍数量竟然比去年还要多。希望可以多水一点竞赛分。

# 科研

随着寒假的 CVPR 的 reject，学长把这篇工作转投了 ECCV，希望这次能有一个好结果。在这篇工作之后，我开始思考的后面的方向，在很多学长潜移默化的影响下。我也开始对具身领域进行一些初步探索。并且靠着这篇在投套磁了 SJTU 的穆尧老师的实验室。受上一篇关于Flow matching在测试时应用 RL 的启发，我开始更多的探索 $\pi$ 系列相关的工作。并且将注意力投射在 post-training 阶段使用 RL 的思路。
